Como usar a pasta /music

1) Copie seus arquivos de música (MP3, M4A, OGG, WAV) para esta pasta.
2) Gere o arquivo index.json com a lista das músicas (em ordem alfabética):
   - Windows: clique duas vezes em gerar-index.bat
   - Mac/Linux: rode `bash gerar-index.sh` no Terminal
3) Faça o deploy para o Netlify (arraste a pasta do site inteira).

Exemplo de index.json:
[
  "minha-musica-1.mp3",
  "outra-faixa.mp3"
]
